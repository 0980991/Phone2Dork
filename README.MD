> [!WARNING]
> This a work in progress and currently is implemented for Dutch phone numbers with '+31' or '06' prefixes.
> Also, Google doesn't like a lot of conditions in their search query and consequently flattens them. This causes results to be suppressed that would otherwise show up if you only search for that specific format.

# Phone2Dork  
A simple python tool for creating a single search query that includes common phone number formats. This allows users to search for specific phone number without manually adjusting the format. Useful for OSINT.

## Getting started 
1. Download repo or zip
2. Run `python main.py -p +31612345678
3. Copy output and paste it in the appropriate search engine
